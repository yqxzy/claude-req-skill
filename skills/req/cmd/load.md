# /req load <id>

**用途**：新对话开始时恢复需求上下文，是跨对话衔接的入口。

**加载文件**：`~/.claude/requirements/<id>-*/` 目录下的 `_state.md`

**执行**：
1. 找到匹配目录，读取 `_state.md`
2. 输出恢复简报（紧凑，不超过 8 行）：

```
📋 [<id>] <需求名>
背景：<背景摘要，一句话>
进度：已完成 Step N，当前在 Step M
选定方案：<方案名> / 未选定
约束：<关键约束，逗号分隔>
上次停留：<_state.md 中的「上次停留」字段>
建议下一步：/req <命令>
```

3. 将需求路径写入 `~/.claude/requirements/_active`
4. **等待用户指令，不主动执行下一步**
